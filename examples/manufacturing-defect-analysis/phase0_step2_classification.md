## 분류 요약

| 행위 유형 | 리소스 |
|----------|--------|
| 탐색(내부) | #1 검사 기준서, #2 공정 매뉴얼, #3 설비 자료, #4 불량 이력 |
| 탐색(외부) | #7 ISO/IATF |
| 접근 | #4 품질 DB (특정 건 조회), #5 MES |
| 실행(유연) | #6 불량 원인 분석 프로세스 |

> **특징:** 실행(확정)이 없다. 이 유즈케이스는 "분석"이라서 고정 절차가 아니라 판단 기반.
> 검사 절차(수입/공정/출하)는 이 유즈케이스 범위 밖 — ALFA에서 2차 유즈케이스로 추가 시 IRM 재돌림.
